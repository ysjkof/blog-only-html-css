근육 길이-장력 관계

근육의 길이가 중간 = 근육이 힘이 가장 강하다.
근육의 길이가 끝범위 = 근육이 힘이 가장 약하다.

근육은 중간범위에 있을 때 힘이 강하고, 끝범위에 있을 때 약합니다. 예를들어, 팔꿈치를 최대한 폈을 때 팔근육의 길이가 5, 최대한 접었을 때가 1이라고 했을 때 근육의 힘은 길이가 3일 때 가장 강하고 1과 5에서 가장 약합니다. 이걸 능동적인 길이-장력 관계라 합니다.

근육길이-장력 관계 = "능동적인 길이-장력 관계" + "수동적인 길이-장력 관계"

그치만 실제로는 3일때나 4일때나 큰 힘의 차이를 못 느낍니다. 왜냐면 3을 지나면서 힘줄, 근막 등의 구조물이 늘어나고 팽팽해지기 때문입니다. 이걸 수동적인 길이-장력 관계라 하고, 능동적인 길이-장력 관계와 수동적인 길이-장력 관계를 합쳐서 근육길이-장력 관계라고 합니다.

수동적인 길이-장력 관계

// 수축성 구조물과 비수축성 구조물로 나눌 수 있습니다.
// 수축성 구조물은 근육인데 스스로 수축할 수 있는 구조물이고,
// 비수축성 구조물은 인대, 힘줄, 근막 같은 스스로 수축할 수 없는 구조물입니다.

해부학 {
    구조물 : {
        수축성 구조물: 근육,
        비수축성 구조물: 힘줄, 근막, 인대 등
    }

노란 고무줄을 양 손가락으로 잡아 당기면 처음에는 아무런 느낌 없이 당겨집니다. 그러다 어느 순간부터 저항감이 생기고 더 늘리면 결국 끊어지죠.

우리 몸에서도 똑같은 일이 일어납니다. 대신 고무줄이 아니고 비수축성 구조물에서 일어납니다.

예를들어, 힘을 뺀 왼쪽 손가락을 오른손으로 펴서 천천히 뒤로 꺾으면 아무 느낌 없다가 어느순간 뻣뻣함이 느껴지고 마지막에는 "아! 여기서 더 꺾으면 병원간다."는 생각이 들겁니다. 건강한 근육이라면 이 뻣뻣함은 비수축성 구조물에서 탄성력이 생기고 팽팽해졌기 때문에 느끼게 됩니다. 이걸 수동적인 길이-장력관계라고 합니다.

- 느슨함이 아무 느낌이 없을 때까지.
- 임계길이는 뻣뻣함이 느껴지는 순간.
- 임계길이를 지나면서 점진적으로 장력이 증가한다.

능동적인 길이-장력 관계

// 수축성 구조물과 비수축성 구조물로 나눌 수 있습니다.
// 수축성 구조물은 근육인데 스스로 수축할 수 있는 구조물이고,
// 비수축성 구조물은 인대, 힘줄, 근막 같은 스스로 수축할 수 없는 구조물입니다.

해부학 {
    구조물 : {
        수축성 구조물: 근육,
        비수축성 구조물: 힘줄, 근막, 인대 등
    }

근육은 중간길이 일 때 힘이 가장 강하고 길어지거나 짧아질수록 약해집니다.

교차연결(crossbridge) = 액틴 + 미오신
교차연결이 많다 = 근육의 힘이 강해진다.
교차연결이 적다 = 근육의 힘이 약해진다.

만약 (근육길이가 중간에서 가까울수록) {교차연결이 많다.}
그외 (근육길이가 중간에서 멀수록) {교차연결이 적다.}

이걸 이해하려면 근수축기전을 알아야 합니다. 근수축기전은 근육이 수축하는 원리를 설명하는 이론입니다. 근육 속에는 액틴과 미오신이라는 것이 많이 있고 액틴과 미오신이 서로 꽉 붙으면 힘이 발생하는데 이걸 교차연결(crossbridge)이라고 합니다. 교차연결이 많을수록 힘이 강해지고 교차연결은 중간범위일 때 가장 많고, 끝으로 갈수록 줄어듭니다.
![[능동적인 길이 장력 곡선.png]]
- 능동적인 길이-장력 관계를 표현한 능동적인 길이-장력 곡선
- A, D = 교차연결 수가 적은 위치.
- A = e.x., 팔꿈치를 완전히 굽힌 상태의 위팔두갈레근.
- D = e.x., 팔꿈치를 완전히 편 상태의 위팔두갈레근.
- B, C = 교차연결의 수가 가장 많은 위치.
- e.x., 팔꿈치 각도 약 90º의 위팔두갈레근.

주의

쉽게 쓰기 위해서 팔꿈치 각도 90º를 교차연결의 수가 가장 많은 위치라고 했습니다. 하지만 이건 팔꿈치의 가동범위가 0º~180º일 때 얘기인데 실제로는 개인마다 이 범위가 다릅니다. 그래서 90º니까 무조건 힘이 가장 강하다고 하면 틀릴 수 있습니다.
